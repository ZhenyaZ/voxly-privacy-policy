Privacy Policy

This privacy policy is applicable to the Voxly app (hereinafter referred to as "Application") for mobile devices, which was developed by Yevhenii Zhelezniakov (hereinafter referred to as "Service Provider") as a Free service. This service is provided "AS IS".

What information does the Application obtain and how is it used?

The Application acquires the information you supply when you download and register the Application. Registration with the Service Provider is not mandatory. However, you may not be able to use some features without registration.

The Application may collect:

  • Account information (email, username, encrypted password)
  • User-created notes and tasks
  • Reminder messages and schedules
  • Voice input (processed for speech-to-text)

What information does the Application collect automatically?

The Application may collect certain information automatically to ensure stability and improve user experience:

  • Device type and unique Device ID
  • Device Push Token (for notifications)
  • Operating system version
  • IP address and general location data (country/city)
  • Usage Data: Interaction with features, screen views, and button clicks.
  • Crash Reports: Stack traces and device state in the event of an application error.

Analytics and Crash Reporting

We use Google Analytics for Firebase and Firebase Crashlytics. These services help us understand how the Application is used and identify technical issues.

  • Anonymized Data: All analytics data is anonymized. We do not use this data to identify individual users personally.
  • Consent-Based: Data collection for analytics purposes only begins after you provide explicit consent via the in-app consent banner.
  • Stability: Crash reports are collected to help the Service Provider fix bugs and prevent future application failures.

Artificial Intelligence (AI)

The Application uses AI technologies (including OpenAI API) to provide core features:

  • Processing voice input for task creation.
  • Automation and smart recommendations based on user activity.
  • Secure processing of text and voice data.

Third Party Services

The Application uses third-party services that may process information:

  • Google Play Services: Essential for Android app functionality.
  • Firebase (Google): For analytics, crash reporting, and backend infrastructure.
  • Expo: Framework used for application development and updates.
  • OpenAI: For AI-powered features.

These providers have their own privacy policies. We encourage you to review them.

User Rights (GDPR)

If you are in the European Economic Area, you have rights including:

  • Access, correct, or request deletion of your personal data.
  • Withdraw Consent: You can disable analytics data collection at any time within the Application settings.
  • Data portability and restriction of processing.
